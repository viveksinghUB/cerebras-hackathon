# Sources

This directory contains raw, immutable source materials about Cerebras.

## What goes here

- PDFs of research papers
- Blog post clippings (saved as `.md`)
- Screenshots and documentation
- Transcripts of talks or demos
- Links collected as `.md` files with metadata
- Any raw material the wiki draws from

## Metadata convention

Each source should have a sibling `*.meta.md` file with:
- **Title:** ...
- **Author:** ...
- **URL:** ...
- **Date:** YYYY-MM-DD
- **Why it matters:** one-sentence note

The agent will create these on first ingest if missing.

## What NOT to do

- Don't edit sources after adding them (immutable layer)
- Don't duplicate content — link to it instead
